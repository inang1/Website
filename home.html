Twitch Tracker Scraper

A webscraper built in Python using Selenium to crawl a particular streamer's Twitch Tracker page and export their stream data. The only inputs needed are the Twitch ID of the streamer along with the export path for the csv.

The data export contains the following columns: Date of Stream, Time of Stream, Duration, Average Concurrent Viewers, Max Concurrent Viewers, Followers, Views, and Title.
