Twitch Tracker Scraper

A webscraper built in Python using Selenium to crawl a particular streamer's Twitch Tracker page and export their stream data. The only inputs needed are the Twitch ID of the streamer along with the export path for the csv.

The csv data export contains the following columns: Date of Stream, Time of Stream, Duration, Average Concurrent Viewers, Max Concurrent Viewers, Followers, Views, and Title.

Twitch Tracker Visualizations (WIP)

Included in this repo are recreations of the Statistics page on Twitch Tracker, all made using the csv export from my Twitch Tracker Scraper. Two versions of the visualizations are included: one made using matplotlib/seaborn with Python and a visual made with PowerBI.

The visualizations include the following:

1. Day of the Week Analysis: Number of Active Days, Number of Active Days by Year, Hours Streamed on Average, Average Number of Viewers, Average number of Followers

2. Statistics by Month: Hours Streamed, Concurrent Viewers, Followers Gained

3. A visualization combining total followers, total views, hours streamed, hours watched, average viewers, and active days.

The Python version also includes functions to pull the following statistics and display them as a table: followers to date, views to date, average stream time, total hours streamed, active days per week, followers per stream, views per stream, average followers gained per hour streamed, and average views per hour streamed. It also can generate a wordcloud of the most common words used in the stream titles.

DDMRP Simulator (WIP)

A DDMRP simulation tool built in Python that takes in a Parts file and Demand History file, cleans and combines them, then simulates the demand history and outputs a results file. Visualizations of the Green/Yellow/Red zones for each part is included with each image having the name of the corresponding part.

When I first started working for Demand Driven Technologies, I would format the data in Excel to match the AI/ML tool format, run it through the AI/ML software multiple times using different settings, format the original data to match a different DDMRP simulation tool within Excel, then combine the different outputs of the AI/ML software and Excel simulation into a results file. All of this was done manually and was very time consuming.

This script automates the process by formatting the client data to the requirements of the AI/ML tool, takes the AI/ML formatted data and runs simulations (removing the need to reformat the data and run it within the Excel simulation tool), then takes the results and outputs them as xlsx files from the tool and combines them for easy comparison. Visualizations of the Green/Yellow/Red zones for each part is available in a separate output folder.
